# 中国式家长 百合 Mod
此 Mod 使女儿可以和儿子版的女同学进行社交，并使女儿可以向女生求婚。

此 Mod 的制作原因是作者不能接受自家养的黄金大白菜被一群奇形怪状的猪拱了而制作，作者对儿子版的各位女主印象颇好并不需要让黄金大猪去拱别的猪，因此不会添加搞基功能也不打算制作搞基 Mod。
